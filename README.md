# Notepads-IP-Lookup

### Installation 
1. Download the .bat file.
2. Run winget install jqlang.jq
3. Run the .bat file.

### Features
Google Maps integration
Coordinates shown inside program once IP is entered
All websites run on the ISP
Checks if there are any leaks on the IP
